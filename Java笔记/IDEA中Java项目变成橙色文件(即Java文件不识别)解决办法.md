# IDEA中Java项目变成橙色文件(即Java文件不识别)解决办法

**IDEA打开项目，里面的文件变成了橙色，本地运行不了**

***解决办法：***

重新设置**moudle**，关闭掉再添加一下，点击**Apply**，项目文件就能识别了，具体操作如下
